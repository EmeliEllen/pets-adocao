# Pets e Adoção

Este projeto é uma página de adoção de pets desenvolvida com HTML e CSS.
O objetivo foi criar um layout simples, organizado e agradável,
com um formulário de contato.

## Tecnologias utilizadas
- HTML5
- CSS3

## O que foi praticado
- Estruturação de layout
- Flexbox
- Estilização de formulários
- Organização de código CSS

## Como visualizar o projeto
Basta baixar os arquivos e abrir o arquivo `index.html`
em qualquer navegador.
